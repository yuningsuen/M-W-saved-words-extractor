# M-W-saved-words-extractor

A python script that exports your saved words on merriam-webster.com to Anki-compatible format, e.g. csv.

# Usage

Simply fill out the `cookies` in main.py with your own MWCOMSESS value, which can be found in the Storage Inspector in your browser's Web Developer Tools(F12) while on the page https://www.merriam-webster.com/.

And then run `python main.py`.
